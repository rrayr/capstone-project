# Personal Finance Tracker

## Description:

Financial Thoughts is a full-stack web application that was designed for users to:

- Register and login using JWT authentication
- Add income and expenses as transactions
- View their transaction history
- Set custom budgets and reminders
- Track and plan ahead their account balance

## Features Include:

- User Authentication
- JWT Authorization
- Protected Routes
- Add, Edit, and Delete Transactions
- Smart Financial Dashboard
- Simulator for Potential Changes
- User-Friendly Design
