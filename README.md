# Personal Finance Tracker

## Description:

Financial Thoughts is a full-stack web application that was designed for users to:

- Register and login using JWT authentication
- Add income and expenses as transactions
- View their transaction history
- Set custom budgets and reminders
- Track and plan ahead their account balance

## Features Include:

- User Authentication
- JWT Authorization
- Protected Routes
- Add, Edit, and Delete Transactions
- Smart Financial Dashboard
- Simulator for Potential Changes
- User-Friendly Design

# Installation:

### Clone Repository

git clone https://github.com/rrayr/capstone-project

cd capstone-project

## Backend

### Setup Backend

cd backend

npm install

### Create .env

DATABASE_URL=your_database_url
SECRET_KEY=your_secret_key
PORT=3001

### Create PostgreSQL Database and run the provided Schema.sql (backend)

### Start backend server 

node server.js OR npm start

## Frontend

### Setup Frontend

cd frontend

### Install Dependencies

npm install

### Start dev server

npm run dev
